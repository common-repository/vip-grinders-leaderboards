== Description ==
This plugin lets you integrate [VIP-Grinders.com](https://www.vip-grinders.com/) [leaderboards](https://www.vip-grinders.com/poker_promotions/) into your website without coding experience.

= Key Features: =
* Easy integration of leaderboards without programming knowledge
* Selection from predefined color schemes
* Mobile compatible
* Styles can be customized according to your own wishes
* [Access to exclusive rake races, chases and leaderboards worth $200k+ per month](https://www.vip-grinders.com/poker_promotions/)
* [Contact us](https://www.vip-grinders.com/contact-us/) to create your own and custom leaderboards

== Installation ==
Just add the plugin in WordPress via "Plugins" -> "Add new". Or upload the plugin to the plugin folder and active the plugin.
